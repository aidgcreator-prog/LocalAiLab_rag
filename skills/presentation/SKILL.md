# Presentation Skill

## Goal
Transform content outlines into professional, visually engaging presentation slide decks in both PowerPoint (PPTX) and PDF formats.

## Workflow
1. **Understand** the presentation goal, target audience, and key messages.
2. **Structure** content into logical sections with clear hierarchy.
3. **Design** slides with clear titles, concise bullet points, and visual elements.
4. **Generate** both PPTX (editable) and PDF (shareable) formats.
5. **Validate** formatting, readability, and professional appearance.

## Best Practices
- One main idea per slide
- Use consistent fonts, sizes, and color schemes
- Keep bullet points short (3-5 words each)
- Add visual breaks with headers and whitespace
- Include title slides, agenda, and closing slides
- Use professional templates and layouts

## Output Format
- PowerPoint file (PPTX) - editable format for customization
- PDF file - shareable, fixed format
- Summary of slides, sections, and design choices

## Tools & Libraries
- `python-pptx` - Create and modify PowerPoint presentations
- `reportlab` or `fpdf2` - Generate PDF presentations
- `Pillow` - Handle images and graphics

## Delivery
- Save files to specified output paths
- Include notes on customization options
- Provide guidance for further editing if needed
