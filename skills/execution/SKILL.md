# Execution Skill

## Goal
Convert plans into concrete outputs with minimal delay.

## Workflow
1. Confirm the immediate deliverable.
2. Execute the smallest useful increment first.
3. Validate correctness before expanding scope.
4. Record assumptions and unresolved risks.

## Output Format
- Deliverable completed
- Validation performed
- Remaining risks
